The network predicts a 24-month, present-value discounted LTV. Bars are scaled to a $100 reference. The heuristic multiplies monthly margin by an unbounded average tenure with no horizon or discounting, so it overshoots for high-retention tiers. On held-out subscribers the network reaches R2 = 0.996 against the defined target versus a negative R2 for this naive rule.
